# MCP Proxy Server for OpenWebUI

A complete MCP (Model Control Protocol) proxy server designed for full compatibility with OpenWebUI. This proxy forwards requests between OpenWebUI clients and model servers while providing authentication, rate limiting, and monitoring.

## Features

- ✅ Full OpenWebUI compatibility
- ✅ Authentication with API keys
- ✅ Rate limiting for request control
- ✅ WebSocket support for real-time communication
- ✅ Health checks and metrics endpoints
- ✅ Error handling and logging
- ✅ Configurable model server URL
- ✅ OpenWebUI specific endpoints
